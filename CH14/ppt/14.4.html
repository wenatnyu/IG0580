14.4 Completing the Square / 完全平方式

1. Intro Problems (10 minutes) / 引入问题（10分钟）

Objective / 目标: To help students understand the concept of completing the square, review quadratic equations, and introduce the need for solving them. / 帮助学生理解完全平方式的概念，复习二次方程，并引入求解的必要性。

Problem 1: Basic Review / 基本回顾

Solve the quadratic equation / 解这个二次方程：

x² + 6x + 5 = 0
  • What type of equation is this? / 这是什么类型的方程？
  • Can students identify it as a quadratic equation? / 学生能识别出它是二次方程吗？
  • Review the general form of a quadratic equation: ax² + bx + c = 0 / 复习二次方程的一般形式：ax² + bx + c = 0

Problem 2: Concept of Completing the Square / 完全平方式的概念

Explain how to transform this equation into a perfect square. / 解释如何将这个方程转化为完全平方式。

x² + 6x + 5 = 0
  1. Take half of the coefficient of x (6), square it: (6/2)² = 9. / 取 x 的系数的一半（6），平方：(6/2)² = 9。
  2. Add 9 to both sides: x² + 6x + 9 = 9 - 5. / 将 9 加到两边：x² + 6x + 9 = 9 - 5。
  3. Result: (x + 3)² = 4. / 结果是：(x + 3)² = 4。

Discussion / 讨论:

  • Why do we add half of 6 squared to both sides? / 为什么将 6 的一半的平方加到两边？
  • What does the left side become? / 左边变成了什么？
  • How do we solve for x? / 如何求解 x？

2. In-Class Practice (20 minutes) / 课堂练习（20分钟）

Objective / 目标: To deepen students' understanding by practicing completing the square on different quadratic equations. / 通过做不同的二次方程练习，加深学生对完全平方式的理解。

Practice 1: Standard Form Quadratic Equations / 标准形式的二次方程

Solve the following equations by completing the square. / 通过完全平方式解以下方程：

  1. x² + 10x - 3 = 0
  2. x² - 4x + 1 = 0

Practice 2: Quadratic Equations with No Constant Term / 不带常数项的二次方程

Solve the following equation. / 解以下方程：

x² + 8x = 0

Practice 3: Challenge Problem / 挑战题

Solve the following equation. / 解以下方程：

x² - 12x + 11 = 0

Tips / 提示: If students struggle, remind them to focus on the coefficient of x and complete the square step by step. / 如果学生有困难，提醒他们注意 x 的系数，并一步一步进行完全平方式的转化。
