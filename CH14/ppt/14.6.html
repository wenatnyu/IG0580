14.6 Factorising Quadratics (a ≠ 1) / 系数不为 1 的二次式因式分解

1. Intro Problems (10 minutes) / 引入问题（10 分钟）

Objective / 目标： To recall factorising when \(a = 1\) and introduce methods for factorising quadratics where \(a \neq 1\). / 复习系数 \(a = 1\) 时的因式分解，并引入系数 \(a \neq 1\) 时的因式分解方法。

Problem 1: Warm-up (a = 1) / 热身：系数为 1

Factorise the quadratic expressions. / 因式分解下列二次式：

  1. \(x^{2} + 7x + 10\)
  2. \(x^{2} - 5x + 6\)

Remind students: “Find two numbers that add to \(b\) and multiply to \(c\).”
提醒学生：寻找“两数相加等于 \(b\)，相乘等于 \(c\)”。

Problem 2: Now a ≠ 1 / 现在 \(a \neq 1\)

Consider the quadratic: / 思考下面的二次式：

2x² + 7x + 3

Guide them through the “AC method” (or “splitting the middle term”). / 通过“AC 法”（拆中项）进行引导。

  1. Write it as \(ax^{2} + bx + c\). Here \(a = 2\), \(b = 7\), \(c = 3\).
    写成 \(ax^{2} + bx + c\) 形式，此处 \(a = 2\), \(b = 7\), \(c = 3\)。
  2. Multiply \(a \times c\): \(2 \times 3 = 6\).
    计算 \(a \times c\)：\(2 \times 3 = 6\)。
  3. Find two integers \(m, n\) such that \(m + n = b\), \(mn = ac\). Here \(m = 6\), \(n = 1\).
    找整数 \(m, n\)，使 \(m + n = b\)，\(mn = ac\)。此处 \(m = 6\), \(n = 1\)。
  4. Rewrite the middle term: \(2x^{2} + 7x + 3 = 2x^{2} + 6x + x + 3\).
    拆中项：\(2x^{2} + 7x + 3 = 2x^{2} + 6x + x + 3\)。
  5. Factor by grouping:
    \(2x^{2} + 6x + x + 3 = 2x(x + 3) + 1(x + 3) = (2x + 1)(x + 3)\).
    分组提取公因式：\(2x^{2} + 6x + x + 3 = 2x(x + 3) + 1(x + 3) = (2x + 1)(x + 3)\)。

Key idea / 关键思想： For \(ax^{2} + bx + c\) with \(a \neq 1\), we can use AC method: multiply \(a \cdot c\), split \(b\) into two numbers, then factor by grouping. / 对于 \(a \neq 1\) 的 \(ax^{2} + bx + c\)，可以用 AC 法：乘 \(a \cdot c\)，拆分 \(b\)，再分组因式分解。

2. In-Class Practice (20 minutes) / 课堂练习（20 分钟）

Objective / 目标： To practise factorising quadratics with \(a \neq 1\), step by step. / 通过循序渐进的练习，掌握 \(a \neq 1\) 时的二次式因式分解。

Practice 1: Positive a and c / \(a, c > 0\)

Factorise the following expressions. / 因式分解下列二次式：

  1. \(2x^{2} + 9x + 7\)
  2. \(3x^{2} + 11x + 6\)
  3. \(4x^{2} + 13x + 3\)

Practice 2: c is negative / 常数项为负

Factorise the following expressions. / 因式分解下列二次式：

  1. \(3x^{2} - 2x - 8\)
  2. \(2x^{2} + x - 6\)
  3. \(5x^{2} - 7x - 6\)

Practice 3: From factorising to solving / 从因式分解到解方程

Solve the following equations by factorising. / 通过因式分解解下列方程：

  1. \(2x^{2} + 5x - 3 = 0\)
  2. \(3x^{2} - 10x + 3 = 0\)

Tips / 提示： Always check by expanding your factors. / 每次因式分解后，用展开检查是否正确。
