14.5 Quadratic Formula / 二次公式

1. Intro Problems (10 minutes) / 引入问题（10分钟）

Objective / 目标: To introduce the quadratic formula and demonstrate how it can be used to solve quadratic equations. / 引入二次公式，演示如何用它来解二次方程。

Problem 1: Quadratic Equation Review / 二次方程复习

Solve the quadratic equation using the quadratic formula: / 使用二次公式解以下二次方程：

x² + 6x + 9 = 0
  • What is the standard form of a quadratic equation? / 二次方程的标准形式是什么？
  • Can students identify the values of a, b, and c? / 学生能识别出 a、b 和 c 的值吗？
  • Apply the quadratic formula: \( x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \). / 使用二次公式：\( x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \)。

Problem 2: Using the Quadratic Formula / 使用二次公式

Solve the quadratic equation: / 解以下二次方程：

2x² + 4x - 6 = 0
  • Identify the coefficients: a = 2, b = 4, c = -6. / 识别系数：a = 2, b = 4, c = -6。
  • Substitute into the quadratic formula. / 将系数代入二次公式。
  • Find the roots of the equation. / 求解方程的根。

2. In-Class Practice (20 minutes) / 课堂练习（20分钟）

Objective / 目标: To practice applying the quadratic formula to different quadratic equations. / 通过做不同的二次方程练习，熟练掌握使用二次公式。

Practice 1: Standard Form Quadratic Equations / 标准形式的二次方程

Solve the following equations using the quadratic formula. / 使用二次公式解以下方程：

  1. x² + 8x + 15 = 0
  2. 3x² - 5x + 2 = 0

Practice 2: Quadratic Equations with No Constant Term / 不带常数项的二次方程

Solve the following equation. / 解以下方程：

x² + 6x = 0

Practice 3: Challenge Problem / 挑战题

Solve the following equation. / 解以下方程：

x² - 4x - 12 = 0

Tips / 提示: If students struggle, remind them to carefully identify a, b, and c and then apply the quadratic formula. / 如果学生有困难，提醒他们仔细识别 a、b 和 c 的值，然后代入二次公式。
